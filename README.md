# ☁️ 오늘

&nbsp;`Spotify API`를 이용한 노래 추천 서비스로<br/>

1. 스포티파이 계정으로 로그인 하기
2. 평소에 자주 듣는 곡 최대 20곡 담기
3. 추천곡 확인하기<br/>

를 통해 스포티파이에서 제공하는 추천곡 리스트를 맛보기 해볼 수 있습니다.

## ✨ 실행방법

1. 클론 후 `npm install`
2. 프로젝트 루트 폴더에 `.env` 파일 생성
3. `NEXT_PUBLIC_SPOTIFY_CLIENT_ID` 입력(`Spotify API`에서 받아야함)
4. `npm run dev`
